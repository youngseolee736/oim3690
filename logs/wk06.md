## AI interaction 3 
- I asked AI how to save the user’s CPA study progress so the data would still be there after refreshing the page. 
- AI explained that I could use `localStorage`to save the data in the browser. It also explained that I need to use `JSON.stringify()`to save arrays and objects, and `JSON.parse()` to load them back. 
- I used this explanation to add local storage to my app. Now, when the user adds a topic, changes a status, or marks something for review, the app saves the data.
